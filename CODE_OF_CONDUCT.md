# Snap-A-Steg Code of Conduct

We want Snap-A-Steg to be a welcoming and safe place for everyone. By participating in this project, you agree to follow these guidelines.

## Our Standards
- Be respectful and courteous to all participants
- No harassment, bullying, or belittling others
- Provide constructive feedback
- Keep discussions relevant to the project

## Reporting Issues
If you encounter behavior that violates this code of conduct, please report it to the maintainers via GitHub issues or email. All reports will be handled confidentially.

## Enforcement
Maintainers may remove content or block users who violate this code of conduct. We reserve the right to take action at our discretion to maintain a safe and welcoming environment.

## Acknowledgment
By participating in this project, you agree to follow this Code of Conduct. Let's work together to make Snap-A-Steg a positive and productive open-source community.
